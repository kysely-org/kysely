# Aliases

You can give an alias for selections and tables by appending `as the_alias` to the name:

import {
  Playground,
  exampleSetup,
} from '../../../src/components/Playground'

import {
  aliases
} from './0040-aliases'

<div style={{ marginBottom: '1em' }}>
  <Playground code={aliases} setupCode={exampleSetup} />
</div>
