Optional connectionTimeoutMillis

connectionTimeoutMillis?: number
  • Defined in src/driver/driver-config.ts:52

Number of milliseconds to wait before timing out when connecting a new connection. By default this is 0 which means no timeout.

Optional idleTimeoutMillis

idleTimeoutMillis?: number
  • Defined in src/driver/driver-config.ts:60

Number of milliseconds a connection must sit idle in the pool and not be checked out before it is disconnected from the backend and discarded. Default is 10000 (10 seconds). Set to 0 to disable auto-disconnection of idle clients

Optional maxConnections

maxConnections?: number
  • Defined in src/driver/driver-config.ts:46

Maximum number of connections the pool should contain by default this is set to 10.
